# Nike Landing Page

A simple Nike landing page built using React and Vite.

## Output

.react-project-1/Screenshots/output.png

## Features

- Navigation bar
- Hero section
- Shop Now and Category buttons
- Amazon and Flipkart icons
- Nike shoe image

## Technologies Used

- React
- Vite
- JavaScript
- CSS

## How to Run

1. Clone the repository.

```bash
git clone <your-repository-url>
```

2. Go to the project folder.

```bash
cd react-project-1
```

3. Install the dependencies.

```bash
npm install
```

4. Start the project.

```bash
npm run dev
```
